# 🔑 구글 마스터 키(Service Account) 발급 및 설정 완벽 가이드

AI가 스프레드시트를 자동으로 만들고 관리하려면 두 가지가 필요합니다: **1. API 켜기**, **2. 키 발급 받기**.
천천히 따라오시면 3분 안에 끝납니다!

### 1단계: 프로젝트 만들기

1. **[구글 클라우드 콘솔](https://console.cloud.google.com/)**에 접속하여 구글 아이디로 로그인합니다.
2. 상단 로고 옆의 **'프로젝트 선택'** (또는 현재 프로젝트 이름) 클릭 -> **'새 프로젝트'** 클릭.
3. 프로젝트 이름: `FamilyApp` (아무거나 상관없음) -> **'만들기'** 클릭.
4. 알림창에서 **'프로젝트 선택'**을 눌러 방금 만든 프로젝트로 이동합니다.

### 2단계: API 켜기 (필수!)

AI가 엑셀과 드라이브를 건드릴 수 있도록 허락해주는 단계입니다.

1. 좌측 메뉴(줄 3개) -> **'API 및 서비스'** -> **'라이브러리'** 클릭.
2. 검색창에 `Google Sheets API` 검색 -> 클릭 -> **'사용'** 버튼 클릭.
3. (중요) 다시 좌측 메뉴 -> **'라이브러리'**로 돌아가서.
4. 검색창에 `Google Drive API` 검색 -> 클릭 -> **'사용'** 버튼 클릭.

### 3단계: 키 발급 받기 (JSON)

AI에게 줄 열쇠를 만드는 단계입니다.

1. 좌측 메뉴 -> **'IAM 및 관리자'** -> **'서비스 계정'** 클릭.
2. 상단 **'서비스 계정 만들기'** 클릭.
   - 이름: `treasure-bot` (아무거나) -> **'완료'** 클릭 (권한 설정 건너뛰기).
3. 리스트에 생긴 이메일 주소(예: `treasure-bot@...`) 클릭.
4. 상단 탭 중 **'키(KEYS)'** 클릭.
5. **'키 추가'** -> **'새 키 만들기'** -> **'JSON'** 선택 -> **'만들기'**.
6. **컴퓨터에 저장된 파일(.json)**이 바로 그 열쇠입니다!

### 4단계: 파일 전달

1. 다운로드된 파일을 `d:\최민수\AI\바이브코딩\FamilyMIssionManger` 폴더로 복사해주세요.
2. 저에게 **"키 복사했어"**라고 알려주시면, 즉시 설치를 시작합니다!
